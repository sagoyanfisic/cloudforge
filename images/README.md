# Images

This directory contains visual assets for CloudForge documentation.

## Files

- `multi-region-example.png` - Multi-Region AWS Architecture example diagram

### How to add the image:

1. Save your multi-region architecture diagram here
2. Name it: `multi-region-example.png`
3. Recommended size: 1200-1400px width, PNG format, <500KB

The image will automatically display in the main README.md.
